# SkelForm Web Player

WIP web player for SkelForm animations.

## Features

Includes all SkelForm runtime features, as well as:

- Animation progress bar
- Switching animations
- Switching styles

## Usage

To be documented.

## Building

Install inliner:

```
npm install -g inliner
```

Then, run `build.py`.

This wil lmake a `dist` folder with the minified version of the web player.
